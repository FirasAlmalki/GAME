# لعبة الكلمة (وضع المحتال)

هذه لعبة ويب بسيطة تتطلب 4 لاعبين أو أكثر. في كل جولة، الجميع يشوفون نفس الكلمة، إلّا واحد منهم (المحتال) اللي يُعرض له "برا السالفة".
المجموعة تحاول تكتشف من هو المحتال من طريقة الكلام أو التصرف، والمحتال يحاول يخفي نفسه.

## كيف تشغل اللعبة على جهازك

1. افتح PowerShell وادخل إلى المجلد:
   ```powershell
   cd "C:\Users\fras1\Desktop\All projects\Game"
   ```
2. نزّل الحزم وابدأ السيرفر:
   ```powershell
   npm install
   npm start
   ```
3. افتح أي متصفح وادخل العنوان:
   ```
   http://localhost:3000
   ```
   أو بدل `localhost` بعنوان الـIP إذا بدك تلعب مع ناس على نفس الشبكة.

### ماذا بعد؟
- أمامك صفحة لعمل الغرف، تنضم لغرفة، وتدخل اسمك.
- كل لاعب يضغط "أنا مستعد".
- لما يكونوا أربعة أو أكثر وجميعهم جاهزين، يبدأ العدّ التنازلي.
- بعد العد، الكل يرى الكلمة ما عدا المحتال اللي يرى "برا السالفة".
- بعد نهاية الجولة يظهر زر "اللعب مرة أخرى". إذا ضغطوا كلهم، تبدأ جولة جديدة.
- إذا أحد طاف الصفحة أو فصل الكومبيوتر، يطلع من الغرفة تلقائي.

## رفع الكود إلى الإنترنت

لو حابب تخلي أصدقاءك يلعبون من أي مكان، انشر المشروع على GitHub وربطه مع Render (أو أي خدمة مشابهة). بعدين استخدم الرابط اللي يعطيك إياه Render وشاركه مع الناس.

الملفات اللازمة فقط هي `server.js` والدليل `public/`، مع `package.json` و`README.md`.

> ⚠️ ملاحظة: حالة الغرف تُخزّن في ذاكرة السيرفر. إذا طفى السيرفر، تمسح كل الغرف والأسماء.

## النقاط الأساسية

- أي مستخدم يمكنه إنشاء غرفة باسم خاص
- تظهر الغرف لجميع الزوار الحاضرين
- عند الانضمام يُطلب اسم اللاعب
- اللاعبون يضغطون "أنا مستعد"؛ بعد أن يكون في الغرفة 4 لاعبين جاهزين، يبدأ العد التنازلي
- في بداية الجولة يُعطى كل لاعب إما كلمة عشوائية أو يظهر له "برا السالفة" إذا كان المحتال
- بعد انتهاء الجولة يظهر زر "اللعب مرة أخرى"؛ عندما يضغط الجميع سيعاد تعيين الحالة وتبدأ جولة جديدة
- الخروج أو غلق الصفحة يخرج اللاعب من الغرفة تلقائياً

> لاحظ: حالة اللعبة محفوظة في الذاكرة على الخادم. عند إعادة تشغيل الخادم تختفي الغرف والأسماء.
