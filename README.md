# لعبة الكلمة

تطبيق ويب بسيط متعدد اللاعبين (4+ لاعبين) حيث يعرف معظم اللاعبين كلمة واحدة ويتخفى "المحتال" الذي لا يعرفها.

## تشغيل المشروع

```powershell
cd "C:\Users\fras1\Desktop\All projects\Game"
npm install
npm start
```

ثم افتح متصفحك على http://localhost:3000

### النشر على GitHub و Render

1. ابدأ مستودعًا جديدًا في المجلد:
   ```powershell
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/yourusername/your-repo.git
   git push -u origin main
   ```
   استبدل الرابط بالرابط الخاص بك.

2. سجّل في [render.com](https://render.com) ثم اختر **New → Web Service**.
   - اربط حساب GitHub واختر المستودع السابق.
   - اترك إعدادات البناء الافتراضية (`npm install && npm start`).
   - تأكد من أنّ `PORT` يُعيّن إلى `3000` افتراضيًا.
   - حدد اسمًا للخدمة ثم `Create Web Service`.

3. بعد البناء ستُعطى لك رابطًا من النموذج:
   `https://<اسم-الخدمة>.onrender.com`.
   شاركه مع أصدقائك، وسيعمل التطبيق مباشرة.

## النقاط الأساسية

- أي مستخدم يمكنه إنشاء غرفة باسم خاص
- تظهر الغرف لجميع الزوار الحاضرين
- عند الانضمام يُطلب اسم اللاعب
- اللاعبون يضغطون "أنا مستعد"؛ بعد أن يكون في الغرفة 4 لاعبين جاهزين، يبدأ العد التنازلي
- في بداية الجولة يُعطى كل لاعب إما كلمة عشوائية أو يظهر له "برا السالفة" إذا كان المحتال
- بعد انتهاء الجولة يظهر زر "اللعب مرة أخرى"؛ عندما يضغط الجميع سيعاد تعيين الحالة وتبدأ جولة جديدة
- الخروج أو غلق الصفحة يخرج اللاعب من الغرفة تلقائياً

> لاحظ: حالة اللعبة محفوظة في الذاكرة على الخادم. عند إعادة تشغيل الخادم تختفي الغرف والأسماء.
